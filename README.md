# Distributed-Dynamic-GNN
Distributed dynamic GNN system, which uses the EvolveGCN (git@github.com:IBM/EvolveGCN.git)
## Run Distributed.py
```
python Distributed.py --config_file ./experiments/parameters_example.yaml
```
